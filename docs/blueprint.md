# **App Name**: Northway

## Core Features:

- Secure User Authentication: Firebase Authentication for secure sign-up, login, and logout using email and password.
- Personalized Onboarding: Multi-step onboarding flow to collect user preferences (profession, study interest, budget, preferred countries) and store them in Firestore.
- Affordable Options Feed: Display a personalized feed of universities matching the user's criteria, sourced from Firestore. Includes the motto Education with purpose, not pressure.
- University Details Display: Show university name, country flag, estimated annual cost, and program highlights in a clear, card-based UI.
- Profile Management: Allow users to edit their profile information and preferences via a dedicated profile settings page.
- Database Integration: Uses Firestore to store user profiles and university data.

## Style Guidelines:

- Primary color: Soft blue (#64B5F6), evoking trust and clarity.
- Background color: Light blue (#E3F2FD), for a clean and open feel.
- Accent color: Deep blue (#1976D2), used sparingly for key actions and highlights to draw user attention without overwhelming the design.
- Body and headline font: 'PT Sans', a humanist sans-serif suitable for headlines and body text.
- Use clean, simple icons from Material Design to represent different categories of universities, countries and costs.
- Card-based layout with rounded corners and subtle shadows for the university feed. Responsive design to support both mobile and desktop.
- Extensive use of animations and transitions to create a beautiful, modern, and highly responsive web app. Examples include parallax scrolling, animated page transitions, and interactive UI elements.
